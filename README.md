# Reto #2

Pues obvio misma instalación jeje
### Instalación

```sh
$ cd oop-backend
$ npm i --save
$ npm run dev
```

Para el uso es necesario un .env con lo siguiente:

```sh
PORT=4000
MONGO_URI=mongodb://localhost:27017/party? 
```

La aplicación correrá por defecto en:

```sh
localhost:4000/api/
```
