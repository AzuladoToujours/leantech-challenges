# Reto #1

### Instalación

```sh
$ cd oop-backend
$ npm i --save
$ npm run dev
```

Sí decide cambiar el puerto a utilizar, sólo es necesario un .env con lo siguiente:

```sh
PORT=3000 
```

La aplicación correrá por defecto en:

```sh
localhost:4000/api/
```
